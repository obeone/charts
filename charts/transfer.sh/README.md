## About

transfer.sh is quick file sharing from the command line: `curl --upload-file`
a file and get a share link back. It supports several storage backends (local
disk, S3, Google Drive, Storj), optional encryption, and previews for common
file types.

- **Application:** [github.com/dutchcoders/transfer.sh](https://github.com/dutchcoders/transfer.sh)
- **Container image:** [`dutchcoders/transfer.sh`](https://hub.docker.com/r/dutchcoders/transfer.sh)
- **Chart source:** [`charts/transfer.sh`](https://github.com/obeone/charts/tree/main/charts/transfer.sh)

## Verifying the chart signature

Charts in this repository are signed with GPG and every release ships a
provenance file. The public key is available at
[`charts.obeone.cloud/public_key.gpg`](https://charts.obeone.cloud/public_key.gpg),
fingerprint `B9FE852F28888D27F8C9A11CD33E04CD22E335CE`.

```shell
# Import the signing key into a legacy keyring (helm verifies with GnuPG v1 keyrings)
curl -fsSL https://charts.obeone.cloud/public_key.gpg | gpg --import
gpg --export > ~/.gnupg/pubring.gpg

# Pull the chart and check it against its provenance file
helm pull --verify obeone/transfer.sh
```
