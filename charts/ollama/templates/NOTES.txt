## Access the API

Port-forward the Ollama API (port 11434) to your machine:

```bash
kubectl port-forward -n {{ .Release.Namespace }} \
  svc/{{ include "bjw-s.common.lib.chart.names.fullname" . }} 11434:11434
curl http://localhost:11434/api/version
```

Pull and run a model:

```bash
curl http://localhost:11434/api/pull -d '{"name":"llama3.2"}'
```
